# 🚀 Internova — Smart Internship & Project Platform

## 📌 Overview  
Internova is a modern web platform where students and developers can showcase their projects, explore internships, and highlight their skills — all in one place.

It acts as a portfolio + opportunity hub, making it easier for recruiters to discover talent efficiently.

---

## ✨ Features  

- 🎯 Portfolio Showcase  
  Display your projects in a clean and professional UI  

- 🔗 Live + GitHub Links  
  Direct access to live demos and source code  

- 💡 Skills Visualization  
  Interactive and modern representation of skills  

- 📱 Fully Responsive  
  Optimized for mobile, tablet, and desktop  

- 🌙 Premium UI/UX  
  Dark blue + white aesthetic with smooth design  

- 📬 Contact Section  
  Easy connection via LinkedIn, GitHub, and Email
