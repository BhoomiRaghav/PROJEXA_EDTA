# 🎨 CSS Folder Documentation

## 📌 Overview  
This folder contains all the styling files used in the Internova project.  
It defines the visual appearance, layout, responsiveness, and overall user experience of the platform.

---

## 🧩 Structure Explanation  

- **style.css**  
  The main stylesheet for the project.  
  It includes global styles, layout design, typography, colors, and component styling.

- **responsive.css** *(if present)*  
  Handles responsiveness for different screen sizes like mobile, tablet, and desktop.

- **components / sections CSS (if any)**  
  Separate files may be used for specific sections like navbar, cards, footer, etc., to maintain modularity.

---

## 🎯 Purpose  

The CSS files are designed to:
- Create a modern and aesthetic UI  
- Ensure consistency across all pages  
- Improve user experience with smooth layouts  
- Support responsive design for all devices  

---

## ⚙️ Features Implemented  

- 🌙 Dark theme (blue + white aesthetic)  
- 🎨 Custom color palette  
- ✨ Animations and transitions  
- 📱 Responsive layouts using media queries  

---

## 💡 Best Practices Followed  

- Clean and organized code structure  
- Reusable classes and components  
- Proper naming conventions  
- Minimal redundancy  

---

## 🚀 Note  

This folder controls the entire look and feel of the project.  
Any UI/UX changes should be handled carefully to maintain design consistency.
